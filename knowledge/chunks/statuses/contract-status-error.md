**Status:** ERROR (Error)

**Business Meaning:** A processing error occurred during contract lifecycle. Check dashboard attention events and status reason for details.

**When This Status Occurs:** Status is set when the contract or amendment reaches this stage in the order lifecycle. Most transitions after submission are driven by Salesforce/Middleware events, not direct dealer action.

**Allowed Actions:** Contact support, Review dashboard alerts

**Restricted Actions:** Most edits

**Possible Next Statuses:** May return to processing states depending on resolution

**Typical User Questions:** What does Error mean? Why is my contract in Error status? How long will it stay in Error?

**Example Support Response:** Your contract is currently in Error status. A processing error occurred during contract lifecycle. Check dashboard attention events and status reason for details. If you need to take action, you can: Contact support, Review dashboard alerts.
